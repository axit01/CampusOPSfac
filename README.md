# AAVISHKAR Faculty Side Portal

Welcome to the newly implemented Faculty Side portal for the AAVISHKAR institutional management system.

## Setup Instructions

1.  **Dependencies**: All necessary dependencies have been installed:
    *   `framer-motion`
    *   `react-router-dom`
    *   `axios`
    *   `socket.io-client`
    *   `lucide-react`
    *   `date-fns`

2.  **Running the Portal**:
    *   Open a terminal in the `2.faculty` directory.
    *   Run `npm run dev` to start the development server.
    *   Ensure the backend server (in `1.admin/server`) is running on `http://localhost:5000`.

## Implemented Features

### 🔐 1. Authentication & Access
*   **Secure Login**: JWT-based session handling with redirection to the dashboard.
*   **Auto-generated Credentials**: Compatible with faculty codes/emails and passwords generated by the Admin panel.
*   **Protected Routes**: Prevents unauthorized access to faculty resources.

### 🏠 2. Faculty Dashboard
*   **Stats Cards**: Real-time overview of Assigned Tasks, Pending Tasks, Upcoming Meetings, and Active Events.
*   **Notification Center**: View recent administrative broadcasts and system alerts.
*   **Quick Actions**: Shortcuts to common management tasks.

### 🎯 3. Task Management
*   **Lifecycle Tracking**: Tasks move through states: Assigned → Accepted → Working → Completed.
*   **Interactive Cards**: Visual progress indicators and status-specific action buttons.
*   **Real-time Updates**: Status changes are immediately synchronized with the database.

### 🤝 4. Collaboration System
*   **Faculty Chat**: Private messaging between registered faculty members.
*   **Admin Broadcasts**: View-only channel for important administrative announcements.
*   **Socket Prepared**: Client-side logic is ready for real-time WebSocket integration.

### 📅 5. Smart Calendar
*   **Custom Monthly View**: High-fidelity grid showing all institutional commitments.
*   **Countdown System**: "Starts In" timers for upcoming meetings and events.
*   **Day Details**: Integrated view of daily tasks and meetings.

### 🏫 6. Visual Timetable
*   **Assigned Lectures**: Personalized grid view of teaching slots.
*   **Role-Based Filtering**: Only shows lectures where the logged-in faculty is the instructor.

### 🎉 7. Events & Meetings
*   **Join Triggers**: Direct links to online meetings (Video Join).
*   **Participation Info**: Detailed priority and location tracking.

### 👤 8. Profile Management
*   **Institutional Identity**: View departmental details and faculty code.
*   **Security Controls**: Update personal info and change access passwords.

## Technology Stack
*   **React 19** with Vite.
*   **Framer Motion** for premium animations.
*   **Vanilla CSS** with a custom design system for high performance and consistent branding.
*   **Lucide React** for modern iconography.
